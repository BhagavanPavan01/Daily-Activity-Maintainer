# Daily Activity Maintainer

A beautiful React application to manage your daily activities with an integrated calendar view, built with Tailwind CSS.

## Features

- ✅ Create and manage daily activities
- 📅 Interactive calendar with color-coded status
- 💾 Automatic localStorage persistence
- 🎨 Beautiful UI with Tailwind CSS animations
- 📱 Fully responsive design
- 🔄 Real-time progress tracking
